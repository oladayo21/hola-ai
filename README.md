# Hola AI

Workshop-Dokumentation: KI-Tools im Coding-Alltag

**Live:** https://hola-ai.dev

## Entwicklung

```bash
make dev      # Lokaler Dev-Server
make build    # Produktion-Build
make deploy   # Deploy zu Cloudflare
```

## Stack

- [Astro](https://astro.build) + [Starlight](https://starlight.astro.build)
- [starlight-theme-next](https://starlight-theme-next.trueberryless.org)
- [Cloudflare Workers](https://workers.cloudflare.com)
